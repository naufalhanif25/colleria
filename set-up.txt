How to set up Colleria:
1. Download and install Python 3 at https://www.python.org
2. Add Python 3 to PATH
3. Download and install Anaconda at https://www.anaconda.com/download
4. Download and install Git at https://git-scm.com/downloads
5. Clone the repository, git clone https://github.com/naufalhanif25/colleria.git
6. Run set-up.bat to install the required libraries.
7. Run run.bat to run Colleria
